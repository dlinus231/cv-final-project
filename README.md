## Project Overview
This project investigates the diagnostic capabilities of CLIP-based vision-language models (specifically QuiltNet-B-32) on the MHIST colorectal polyp dataset. We augment the dataset with descriptive captions generated by CONCH, a state-of-the-art vision-language foundation model. Unlike standard CLIP-style models, CONCH incorporates a generative decoder via the CoCa framework, enabling it to produce free-form pathological descriptions of histology images directly from visual features. These captions are then used as text supervision for fine-tuning QuiltNet on the MHIST classification task.

## Quick Start: Interactive Demo

To see the final model's diagnostic performance and self-attention reasoning in action:

1.  **Clone & Setup:**
    ```bash
    git clone https://github.com/dlinus231/cv-final-project/
    cd computer_vision_project
    ```

2.  **Install Dependencies:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    pip install -r quilt_notebooks/requirements.txt
    ```

3.  **Run the Demo:**
    - Open **`quilt_notebooks/model_demo.ipynb`**.
    - Run the notebook cells to load the best multimodal weights discovered via Bayesian optimization.
    - The notebook will automatically sample random patient cases and display:
        - The pathology image.
        - **Vision Attention Maps**: Spatial regions the model prioritized.
        - **Text Attention Weights**: Semantic tokens in the clinical prompt that influenced the diagnosis.
